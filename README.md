A basic computer vision project: a real-time hand tracking system using Mediapipe and OpenCV.
The program detects hand landmarks (key points of fingers and connections) and draws them on the live webcam feed. It also calculates and displays the Frames Per Second (FPS) to measure performance.

Features

Real-time hand detection using the webcam

Display of hand landmarks and skeleton connections

FPS calculation and display

Exit the program easily by pressing Q

Technologies

Python

OpenCV
 – image processing and display

Mediapipe
 – hand tracking solution
