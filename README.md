<h2>Description</h2>

<p>
A basic computer vision project that implements a real-time hand tracking system using <b>Mediapipe</b> and <b>OpenCV</b>. The program detects hand landmarks (key points of fingers and connections) and draws them on the live webcam feed. Frames Per Second (FPS) are calculated and displayed to measure performance.
</p>

<ul>
  <li>Real-time hand detection using webcam</li>
  <li>Display of hand landmarks and skeleton connections</li>
